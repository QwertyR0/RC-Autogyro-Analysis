# (EN) RC Autogyro Hemistatic Analysis

Following the excel sheet, "RC Auto Gyro Calculator" by Pieter Mulder, this code implements the calculations in a dynamic way for analyzing the performance of RC autogyros in autorotation. Some of the Dynamic calculations; Lift efficiency related to Rotor Area and Rotor Material Density, Angle of Attack related to Rotor RPM Speed and Lift Coefficient.

This codebase will be a much simpler alternative to the gyroblade (excessive) flap calculator by Moore Penrose.

# (TR) RC Otogiroların Yarı-statik Analizi

Pieter Mulder tarafından hazırlanan “RC Auto Gyro Calculator” adlı Excel tablosunu temel alan bu kod, RC otogiroların otorotasyon sırasındaki performansını analiz etmek için hesaplamaları dinamik bir şekilde gerçekleştirir. Dinamik hesaplamalardan bazıları şunlardır: Rotor alanı ve rotor malzeme yoğunluğuna bağlı kaldırma verimliliği, rotor devri ve kaldırma katsayısına bağlı hücum açısı.

Bu yazılım, Moore Penrose tarafından geliştirilen gyroblade (eksesif) esneme hesaplayıcısına göre çok daha basit bir alternatif olacaktır.
